Social Buttons for Twitter Bootstrap

Gorgeous, vector, Retina-ready, and customisable social media buttons using Bootstrap and Font Awesome.

Usage


is just 1 line of HTML code:

<button class="btn btn-facebook"><i class="icon-facebook"></i> | Connect with Facebook</button>

Installation

Full instructions can be found here, but basically:

  1. Clone the git repo
  2. Add social-buttons.css to your app (or import social-buttons.less into your bootstrap.less file)
  3. Use buttons!

Requirements

  • Bootstrap 2.3.1 (older versions may work)
  • Font Awesome 3.0.2 - if you want icons inside the buttons
