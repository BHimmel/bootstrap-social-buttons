Example usage


is just 1 line of HTML code:

<button class="btn btn-facebook"><i class="icon-facebook"></i> | Connect with Facebook</button>

Quick installation

  1. Download social-buttons.css and include in your project. Done!

Installation

Full instructions can be found here, but basically:

  1. Navigate to the Github project.
  2. Add social-buttons.css to your app (or import social-buttons.less into your bootstrap.less file)
  3. Use buttons!

Requirements

  • Bootstrap 2.3.1 or newer (older versions may work)
  • Font Awesome 3.0.2 or newer - if you want icons inside the buttons
